🚀 FASTEST DEPLOYMENT METHOD:

1. Extract the deployment package:
   tar -xzf firemon_deployment.tar.gz

2. Upload to your web server:

   Option A - FTP/SFTP:
   • Open FileZilla or similar FTP client
   • Connect to your server
   • Navigate to /public_html/firemon/ (or wherever firemon.netsudo.com points)
   • Upload: index.html, device_data_combined.js, .htaccess

   Option B - cPanel:
   • Login to cPanel
   • File Manager → Navigate to firemon directory
   • Upload → Select the 3 files

   Option C - Command Line (SSH):
   scp index.html device_data_combined.js .htaccess user@server:/path/to/firemon/

3. Test the site:
   Open: https://firemon.netsudo.com

🔒 SECURITY SETUP (HIGHLY RECOMMENDED):

Since this contains sensitive device information, ADD PASSWORD PROTECTION:

Step 1: Edit .htaccess file, uncomment these lines:
   AuthType Basic
   AuthName "FortiGate Inventory - Authorized Personnel Only"
   AuthUserFile /full/path/to/.htpasswd
   Require valid-user

Step 2: Create password file (on server via SSH):
   htpasswd -c /path/to/.htpasswd admin
   (Enter password when prompted)

Step 3: Update AuthUserFile path in .htaccess with full server path

🆘 TROUBLESHOOTING:

Problem: Blank page
→ Check browser console (F12) for errors
→ Verify both files are uploaded

Problem: "DEVICE_DATA undefined"
→ Ensure device_data_combined.js is in same directory
→ Check file permissions (should be 644)

Problem: Charts not showing
→ Check internet connection (Chart.js loads from CDN)
→ Clear browser cache (Ctrl+Shift+R)
